Windowsアプリが接続準備を行い、Chrome拡張機能の導入画面へ案内します。拡張機能の追加は、利用者自身が確認して行います。

    Windowsアプリ

    Fuzzyをダウンロード

  2. 2

    Chrome拡張機能

    案内に沿って拡張機能を追加

    初回起動後に表示される案内から導入画面を開き、内容を確認して追加します。

    Moodle

    Fuzzyタブから利用開始

    アプリとの接続を確認したら、Moodle上に表示されるFuzzyタブから機能を開きます。

安全性とプライバシー

あなたの資料は、
あなたのPCを中心に。

Fuzzyはローカルでの利用を基本とし、授業資料の整理を支援します。自動で勝手にファイルを移動・削除するのではなく、候補を示して利用者の判断を待ちます。

  • ✓
    利用者が最終決定保存先や操作内容を確認してから進めます。
  • ✓
    ローカル中心の設計資料と学習情報を、このPCを中心に扱います。
  • ✓
    権限を分かりやすく説明拡張機能が必要とする権限と目的を案内します。

よくある質問

導入前の疑問に
答えます。

今すぐダウンロードできますか？

現在は開発・レビュー段階です。配布可能な版が公開されると、GitHub Releasesにインストーラーと更新内容が表示されます。

拡張機能は自動で追加されますか？

無確認で追加はしません。Windowsアプリから導入先を開き、Chromeに表示される内容を確認して利用者が追加します。

保存した資料は勝手に移動されますか？

Fuzzyは保存先や整理方法を提案します。保存済みファイルの移動や削除は、利用者への提示・確認を前提とします。

不具合や要望はどこへ送ればよいですか？

GitHub Issuesから、再現手順や画面の状態を添えて報告できます。
